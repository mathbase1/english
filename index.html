    What this build now enforces

    Will not over-reward

    • long or confident writing on its own
    • basic quotation use being called “judicious”
    • weak everyday comments like “bad”, “pretty”, or “smells bad” being treated as strong analysis
    • coverage without analysis
    • clear explanation being mistaken for perceptive insight
    • coverage or fluency being mistaken for sustained conceptual analysis
    • feature spotting without precise meaning

    Will still allow

    • short but valid analysis
    • quote-then-explain structure across adjacent sentences
    • grounded inference such as fantasy, claustrophobia, dehumanisation, and alienation
    • some generic GCSE phrasing if the actual analysis is good enough

    Level 4 only if

    • analysis is consistently accurate and grounded
    • there are multiple developed or perceptive units
    • evidence is selective rather than merely relevant
    • a conceptual thread is sustained across the answer
