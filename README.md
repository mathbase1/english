# GCSE English GitHub Pages Quiz

This is the GitHub Pages version of the GCSE English self-marking quiz.

## Files

- `index.html` – the page students open
- `styles.css` – the styling
- `config.js` – the extract, question, and indicative content you can swap easily
- `app.js` – the browser-based marking logic

## What to edit

Open **`config.js`** and change:

- `questionPrompt`
- `sourceExtract`
- `indicativeContentGroups`
- `highValueReferences`
- `salientExtractTerms`

That is the main teacher-editable area.

## How to publish on GitHub Pages

1. Create a new GitHub repository.
2. Upload **all files in this folder** to the root of the repository.
3. On GitHub, open the repository.
4. Go to **Settings → Pages**.
5. Under **Build and deployment**, choose **Deploy from a branch**.
6. Choose your branch, usually **main**, and choose the root folder **/**.
7. Save it.
8. Wait for GitHub to publish the site.

After that, the student link will normally look like:

`https://YOUR-USERNAME.github.io/YOUR-REPOSITORY-NAME/`

## Notes about marking

This is a **rule-based** marker. It is not AI and it is not identical to a human examiner.

It looks for:

- coverage of indicative content
- quotations / extract references
- analytical verbs
- language-method terms
- sentence / punctuation comments
- developed explanation
- simple SPaG warning signs
- overly copied responses

## Suggested use

This works best as:

- a self-check tool
- a revision quiz
- a low-stakes practice marker
- a classroom feedback tool

For final exam judgement, a teacher should still moderate the answer.
